# Portfolio Projects

## Overview
This section showcases end-to-end AI-SRE projects that apply your learning to Kubernetes challenges. Projects like AI-driven autoscalers or anomaly detection bots demonstrate your skills to employers and the community.

## Learning Objectives
- Build complete AI-SRE projects for Kubernetes.
- Document projects with clear READMEs and code.
- Share projects to build your professional brand.

## Key Concepts
- **Project Design:** Combine AI, MLOps, and Kubernetes for practical solutions.
- **Documentation:** Write READMEs for reproducibility.
- **SRE Application:** Solve real-world problems like autoscaling or monitoring.

## Project Ideas
1. **Predictive Autoscaler:**
   - **Description:** Use KubeFlow to deploy an ML model that predicts pod scaling needs based on Prometheus metrics.
   - **Tools:** KubeFlow, Prometheus, Python, Minikube.
   - **Directory:** `predictive_autoscaler/`
   - **Deliverables:** Model code, KubeFlow pipeline, README.
2. **Anomaly Detection Bot:**
   - **Description:** Build a Slack bot using LangChain to detect and report Kubernetes anomalies.
   - **Tools:** LangChain, Prometheus, Slack API.
   - **Directory:** `anomaly_detection_bot/`
   - **Deliverables:** Bot script, integration guide, README.

## Hands-On Exercises
1. **Predictive Autoscaler Project:**
   - Train a model with Prophet and deploy via KubeFlow.
   - Save code in `predictive_autoscaler/model.py` and pipeline in `predictive_autoscaler/pipeline.yaml`.
2. **Anomaly Detection Bot:**
   - Build a bot to query logs and alert on anomalies.
   - Save as `anomaly_detection_bot/bot.py`.
